# Azure Multi-Agent Trend Analysis System

## Overview

This project demonstrates a production-grade multi-agent AI system using:

- Azure OpenAI
- Azure-compatible web search
- Agent-to-Agent orchestration (A2A-style)

## Agents

- Trending Topics Agent
- Trend Analyzer Agent
- Host Orchestrator Agent

## How to Run

1. Set `.env`
2. Install dependencies
3. Run:
   ```bash
   python run_all.py
   ```
